Download Details

This file download service is an alternative to obtaining the file from the studio. It is not endorsed or supported by the studio. However, it provides convenience to you as a volunteer.

You don’t need to visit the studio or search for the file on the shared drive that became available during lockdowns. This service provides you with a convenient way to obtain the file.

The file is captured by software created around 2010 and runs on the creator’s home computer. The software automates some processing of the file to bypass steps that are manually performed, such as converting to monaural, adding metadata and downscaling the bitrate.
